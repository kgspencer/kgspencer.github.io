This is a web-based Cryptoquip game that I've been working on for the past few weeks. It's similar to Wordle in that a new cryptoquip will be available every day.

There are a few behaviors limited by the scope of this website, which include:

  • Your game progress will not be saved if you refresh or exit the page.
  • The game only accepts input from the on-screen keyboard, not your physical one.
  • Though I tried to test the game, there may be errors on my end. Let me know if something isn't working the way it's supposed to!

I hope you enjoy this game!

Kathryn

This game is meant to mimic the classic Cryoptoquip game. The given phrase is encrypted with a letter substitution cipher. The goal is to decode the phrase by figuring out which letter each encrypted letter represents.

Hovering over a letter will highlight each same letter, so you can see the placements of each character. Clicking on a letter will select it. It will look like this:

Q

To swap a letter, you can choose a letter from the keyboard. Once a letter from the phrase and the keyboard have been selected, the phrase letter will become locked. You can still swap a locked letter, but it will not appear highlighted anymore when you hover over it. Once locked, the letter will look like this:

Q

When you think you've solved the phrase, you can test your answer using the Check Answer button.

Notes:

  • The timer will start once you have swapped your first letter.
  • The 'Selected Letter' on screen refers to the phrase letter you have selected. It is useful if you're swapping a locked letter.
  • You cannot swap a letter that has already been swapped. Try swapping the desired letter with a different one first.
  • The order in which you select a phrase letter and keyboard letter should not matter. Note that if you choose from the keyboard first, the phrase letter will be locked upon clicking it.
  • You shouldn't be able to swap any punctuation or spaces, and their appearance shouldn't change.
